## v0.11.4

### English

#### Highlights

- Added a bilingual Web UI with built-in English and Simplified Chinese switching.
- The Web UI now remembers the selected language locally and applies it more consistently across navigation, CLI takeover, services, and system status views.

#### What Changed

- Added: Web UI language switching between English and Simplified Chinese
- Added: local language preference persistence in the Web UI via browser storage
- Improved: major management surfaces, including navigation, provider flows, CLI takeover, services, and system status, now have broader Chinese localization coverage
- Improved: language selection now exposes proper radio-style accessibility semantics and updates document language metadata more accurately on load
- Fixed: localized provider key helper text no longer hardcodes English pluralization rules
- Changed: removed brittle UI implementation-detail tests and kept server-side route and access-control coverage focused on meaningful backend behavior

### 中文

#### 亮点

- Web UI 现在支持中英文双语切换。
- 管理界面会在本地记住你选择的语言，并且在导航、CLI 接管、服务、系统状态等页面上更完整地应用对应语言。

#### 变更内容

- 新增：Web UI 支持 English 与简体中文切换
- 新增：Web UI 通过浏览器本地存储记住语言偏好
- 改进：导航、Provider 管理、CLI 接管、服务、系统状态等主要界面的中文覆盖更完整
- 改进：语言切换器补充了更合适的单选可访问性语义，并在页面加载时更准确地更新文档语言元数据
- 修复：Provider key 数量提示不再硬编码英文复数规则
- 调整：删除脆弱的 UI 实现细节测试，保留更有意义的服务端路由与访问控制覆盖
